# Your images folder

**Path:** `frontend/public/assets/images/`

Drop any `.jpg` / `.jpeg` / `.png` / `.webp` here. The site **auto-detects** them when you start or refresh `npm run dev` — no need to edit code for each new file.

## How they are used

| Priority | Filename hints | Used for |
|----------|----------------|----------|
| First | `hero` in name | Hero slider (first), main brand photo |
| | `intro` in name | About page + intro section |
| | `kitchen`, `bedroom`, `living` | Intro / gallery |
| | `bathroom` | Gallery + hero slides |
| | `office`, `commercial` | Services page header |
| All files | any name | Hero slider (up to 6) + **Our work** gallery |

## Flyers

**Path:** `frontend/public/assets/fliers/` (spelling **fliers**)

- `flyer-1.jpg`, `flyer-2.jpg`, etc.

Restart dev server after adding new files if they do not show immediately.
